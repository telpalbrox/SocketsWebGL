WebSockets + WebGL

Introducción

Vamos a hablar sobre WebSockets y WebGL. Nuestro proyecto ha consistido en a partir de un ejemplo hecho en WebGL implementar sockets para que el ejemplo se pueda retransmitir en tiempo real a muchos navegadores a la vez.

Sockets

Empezaré hablando sobre sockets y como los he implementado en el navegador y servidor con el lenguaje javascript.

¿Qué es un socket?

Según Wikipedia:

Socket designa un concepto abstracto por el cual dos programas (posiblemente situados en computadoras distintas) pueden intercambiar cualquier flujo de datos, generalmente de manera fiable y ordenada.

¿Qué quiere decir esto?

Pues básicamente un socket es como un túnel que enlaza varias instancias de tu programa. Todo lo que introduzcas en el socket puede llegar a todos los clientes. El socket garantiza que llegará bien y de forma ordenada.

¿Por qué usar sockets en el navegador?

Los sockets son necesarios principalmente cuando se necesita una comunicación en tiempo real y de baja latencia.

Se suelen usar para juegos multijugador, chats, actualizaciones en tiempo real, etc...

¿Cómo se usan los sockets en el navegador?

En el navegador los sockets están implementados a través de WebSockets. Este tipo de sockets están implementados sobre un único socket TCP que puede enviar y recibir datos de forma simultánea.

Para usarlos en el navegador HTML 5 dispone de una API que incluye los WebSockets y que se puede usar a través de Javascript.

En nuestro caso vamos a usar una librería llamada Socket.io que funciona sobre la API de HTML 5 en JavaScript.

¿Cómo se implementan los WebSockets de lado del servidor?

El socket del navegador necesita conectarse a un servidor para ir enviando y recibiendo los distintos mensajes. Generalmente este servidor es el mismo que le proporciona la página web.

En el lado de servidor hay muchas más opciones. Hay librerías disponibles para los principales lenguajes de programación: JavaScript, Java, .NET, PHP, Python, Ruby, etc.

En mi caso elegí la librería Socket.io para JavaScript que está implementada sobre Node.js.
